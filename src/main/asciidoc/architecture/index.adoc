[[the-robot-architecture]]
The robot architecture
----------------------

[[introduction]]
Introduction
~~~~~~~~~~~~

A robot `architecture` primarily refers to the software and hardware
framework for controlling the robot.

Every robot has the following components:

* Sensors
* Actuators
* Control Unit
